# Pili - 隔壁过屏式随机视频播放器

> 仿B站风格全屏视频播放器，支持本地+远程视频源，随机下一集

## 特性

- **B站风格界面** — 底部控制栏 + 进度条 + 音量滑块
- **PC全屏** — 自动隐藏控制栏，鼠标滑出显示
- **移动端竖屏/横屏** — 默认竖屏全屏，左上角横屏按钮
- **随机下一集** — 就像看电视换台一样
- **本地+远程** — 挂载本地目录或连接远程API
- **模式切换** — 循环播放 / 单曲播放 / 自动连播

## 快捷键

| 键 | 功能 |
|---|------|
| `Space` / `K` | 播放/暂停 |
| `N` / `→` | 随机下一个 |
| `P` / `←` | 上一个 |
| `F` | 全屏 |
| `S` | 切换源 |
| `M` | 循环/单曲切换 |
| `V` | 静音开关 |
| `L` | 横屏 (移动端) |

## Docker

```bash
docker run -d --name pili -p 8080:8080 \
  -v pili-data:/data \
  -v /你的/视频目录:/videos:ro \
  lzylipu/btv:latest
```
